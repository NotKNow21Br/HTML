Funzioni

Concetti Base

Una funzione è una relazione tra due insiemi (A → B) che ad ogni elemento di A associa uno solo elemento di B.

Terminologia:

  • x = Variabile indipendente
  • y = f(x) = Variabile dipendente
  • Dominio di f = Insieme (es: A)
  • Codominio di f = Insieme (es: B)
  • y = Immagine di x
  • x = Controimmagine di y

Esempio: Y = x²

  • f: ℝ → ℝ
  • Dominio di f = Insieme di tutte le x
  • Codominio di f = Insieme di tutte le y
  • Immagine di f = Insieme di tutti i valori f(x)

Proprietà delle Funzioni

1. Funzioni Iniettive

Una funzione è iniettiva quando ad ogni elemento di B corrisponde al più un elemento di A.

Esempio: f: ℝ → ℝ

  • f(x) = 3x è iniettiva
  • f(x) = x² non è iniettiva

2. Funzioni Suriettive

Una funzione è suriettiva quando ogni elemento di B è immagine di almeno un elemento di A.

Esempio: f: ℝ → ℝ

  • f(x) = 3x è suriettiva
  • f(x) = x² è suriettiva solo se consideriamo B = [0,+∞)

3. Funzioni Biunivoche (o Biettive)

Una funzione è biunivoca quando è sia iniettiva che suriettiva.

Esempio: f: ℝ → ℝ

  • f(x) = 3x è biunivoca
  • f(x) = x² non è biunivoca

Funzioni Inverse

Le funzioni iniettive sono invertibili e la funzione inversa si indica con f⁻¹.

Per trovare la funzione inversa:

  1. Sostituire f(x) con y
  2. Scambiare x e y
  3. Risolvere rispetto a y

Esempio: f(x) = 3x

  • y = 3x
  • x = 3y
  • y = x/3
  • Quindi: f⁻¹(x) = x/3

Funzioni Pari e Dispari

Una funzione si dice pari quando:

f(-x) = f(x)

Una funzione si dice dispari quando:

f(-x) = -f(x)

Esempi:

  • f(x) = x⁴ - 2x² + 3 è pari
  • f(x) = |x| è pari
  • f(x) = x³ - 5x è dispari

Graficamente:

  • Funzione pari: simmetrica rispetto all'asse y
  • Funzione dispari: simmetrica rispetto all'origine

Equazioni

Equazioni Irrazionali

Per risolvere equazioni con radicali:

  1. Verifica che il radicando sia ≥ 0
  2. Eleva entrambi i membri all'esponente appropriato
  3. Risolvi l'equazione risultante
  4. Verifica le soluzioni nel dominio originale

Esempio:

√(2x² - 3x + 4) = x

Condizioni:

  • 2x² - 3x + 4 ≥ 0 (sempre verificata)
  • Elevando al quadrato: 2x² - 3x + 4 = x²
  • x² - 3x + 4 = 0

Equazioni con Modulo

Per equazioni del tipo |f(x)| = k:

  • Se k < 0: nessuna soluzione
  • Se k = 0: f(x) = 0
  • Se k > 0: f(x) = k oppure f(x) = -k

Esempio 1: |3 - 5x| = 4

  • 3 - 5x = 4 oppure 3 - 5x = -4
  • -5x = 1 oppure -5x = -7
  • x = -1/5 oppure x = 7/5

Esempio 2: |x² - 4x + 3| = 2

  • x² - 4x + 3 = 2 oppure x² - 4x + 3 = -2
  • x² - 4x + 1 = 0 oppure x² - 4x + 5 = 0
  • Risolvere le equazioni di secondo grado

Disequazioni

Disequazioni Irrazionali

Per risolvere disequazioni con radicali:

  1. Verifica il dominio (radicando ≥ 0)
  2. Se l'indice è pari, considera che il radicale è sempre ≥ 0
  3. Risolvi considerando il segno del radicando

Esempio:

√(x² + 5x + 6) > 2

Si risolve:

  • x² + 5x + 6 ≥ 0 (dominio)
  • (x² + 5x + 6) > 4 (elevando al quadrato)
  • x² + 5x + 2 > 0

Disequazioni con Modulo

Caso particolare: |f(x)| < k o |f(x)| > k

  • Se k < 0: impossibile per |f(x)| < k
  • Se k = 0: f(x) = 0
  • Se k > 0: -k < f(x) < k per |f(x)| < k

Esempio 1: |x² + 5x + 6| < 2

  • -2 < x² + 5x + 6 < 2
  • x² + 5x + 4 < 0

Esempio 2: |x² + 5x + 8| > 0

  • x² + 5x + 8 > 0 oppure x² + 5x + 8 < 0
  • Risolvere il sistema di disequazioni

In Generale: |f(x)| ≷ g(x)

Si possono presentare due casi:

  • Se g(x) ≥ 0: f(x) > g(x) o f(x) < -g(x)
  • Se g(x) < 0: sempre vera

Esempio: |2-x| > 4+3x

  • Studiare il segno di 4+3x
  • 2-x > 4+3x oppure 2-x < -(4+3x)
  • Risolvere il sistema risultante

La Retta

Equazione della Retta

Forme dell'equazione:

  • Forma esplicita: y = mx + q
    • m = coefficiente angolare (pendenza)
    • q = termine noto (intercetta y)
  • Forma implicita: ax + by + c = 0

Casi Particolari:

  1. Rette parallele all'asse x:
    • y = k → funzione costante
    • Caso particolare: y = 0 → asse x
  2. Rette parallele all'asse y:
    • x = k → non è una funzione
    • Caso particolare: x = 0 → asse y
  3. Rette passanti per l'origine:
    • y = mx → termine noto = 0
    • Il coefficiente m dipende dall'inclinazione della retta

Parallelismo

Due rette sono parallele se e solo se hanno lo stesso coefficiente angolare:

r₁: y = mx + q
r₂: y = m'x + q'
Parallele ⟺ m = m'

Condizione di parallelismo:

  • m = m' (coefficienti angolari uguali)
  • q ≠ q' (termini noti diversi)

Perpendicolarità

Due rette sono perpendicolari se e solo se i loro coefficienti angolari sono uno l'opposto del reciproco dell'altro:

m₁ · m₂ = -1

Per rette perpendicolari:

  • m' = -1/m
  • Condizione di perpendicolarità: m · m' = -1

Posizioni Reciproche di Due Rette

  • Parallele: m = m' e q ≠ q'
  • Coincidenti: m = m' e q = q'
  • Incidenti (1 punto di intersezione): m ≠ m'
  • Perpendicolari (caso particolare di incidenti): m · m' = -1

Distanza Punto-Retta

Per una retta in forma implicita ax + by + c = 0 e un punto P(x₀, y₀), la distanza d è:

d = |ax₀ + by₀ + c| / √(a² + b²)

Per una retta in forma esplicita y = mx + q:

d = |mx₀ - y₀ + q| / √(1 + m²)

Nota: La distanza è sempre positiva o nulla.
